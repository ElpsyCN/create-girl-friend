> 用你喜爱的语言创建属于你的对象（女朋友）。

## Contribute Guide

- 将你的代码放置在对应 `languages` 中的语言目录下
- 对你的代码的功能与使用方式进行描述（必要时在代码中添加注释）
- 你可以合理地重构前人代码（不必束手束脚）
- commit 信息请遵循 [gitmoji 规范](https://gitmoji.carloscuesta.me/)
- 善用 `.gitignore` 忽略你的语言中不应提交至本仓库的构建产物
- 尽可能地让你的女朋友生动复杂有趣但又不失去控制吧
